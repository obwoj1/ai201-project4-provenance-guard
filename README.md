# Provenance Guard

**AI201 Project 4 | Student: obaw**

A Flask API that detects whether a piece of text was written by a human or
generated by AI. It combines a **Groq LLM signal** with **pure-Python
stylometric heuristics**, records every decision to an audit log, and supports a
basic appeals workflow.

📄 See [planning.md](planning.md) for the design rationale.

---

## How it works

Each submission is scored by two independent signals:

1. **Groq LLM (60%)** — `llama-3.3-70b-versatile` returns a JSON score from 0.0
   (human) to 1.0 (AI).
2. **Stylometric heuristics (40%)** — sentence-length variance, type-token
   ratio, and punctuation density, computed with no external libraries.

The signals are combined into a confidence value and mapped to a label:

| Confidence | Attribution | Label |
|---|---|---|
| ≥ 0.70 | `likely_ai` | ⚠️ Likely AI-Generated |
| 0.40–0.69 | `uncertain` | 🔍 Uncertain Origin |
| < 0.40 | `likely_human` | ✅ Likely Human-Written |

## Setup

```bash
# 1. Create and activate a virtual environment
python3 -m venv .venv
source .venv/bin/activate

# 2. Install dependencies
pip install -r requirements.txt

# 3. Configure your Groq API key
cp .env.example .env
# then edit .env and set GROQ_API_KEY=your_actual_key

# 4. Run the server
python3 app.py
```

The server starts on `http://127.0.0.1:5000`.

## API

### `POST /submit`

Classify a piece of text. Rate limited to **10 per minute / 100 per day**.

**Request**
```json
{ "text": "The text to analyze...", "creator_id": "user-123" }
```

**Response**
```json
{
  "content_id": "…uuid…",
  "attribution": "likely_ai",
  "confidence": 0.78,
  "label": "⚠️ Likely AI-Generated",
  "groq_score": 0.81,
  "stylo_score": 0.72
}
```

Returns `400` if `text` or `creator_id` is missing.

**Example**
```bash
curl -X POST http://127.0.0.1:5000/submit \
  -H "Content-Type: application/json" \
  -d '{"text": "Your sample text here.", "creator_id": "user-123"}'
```

### `POST /appeal`

Flag a prior classification for human review.

**Request**
```json
{ "content_id": "…uuid…", "creator_reasoning": "This was written by me." }
```

**Response**
```json
{
  "message": "Appeal received. Your submission is now under review.",
  "content_id": "…uuid…"
}
```

Returns `404` if the `content_id` is not found.

### `GET /log`

Return the full audit log.

**Response**
```json
{ "entries": [ { "content_id": "…", "status": "classified", ... } ] }
```
